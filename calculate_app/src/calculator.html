Инструкция для Calculator

Базовый калькулятор

Основные функции калькулятора включают:

  • Скобки (()): позволяют изменять стандартный приоритет выполнения операций;
  • Сложение (+): Функция для складывания двух чисел. Также присутствует обработка унарного плюса;
  • Вычитание (-): Функция для вычитания одного числа из другого. Также присутствует обработка унарного минуса;
  • Умножение (*): Функция для умножения двух чисел;
  • Деление (/): Функция для деления одного числа на другое. При делении на ноль ответом будет бесконечность;
  • Возведение в степень (^): Функция для возведения в степень одного числа другое;
  • Остаток от деления (mod): Функция для нахождения отстатка от деления одного числа на другое;
  • Присутствует обработка выражения с опущенным знаком `*`
  • Ввод чисел: Возможность пользователю вводить числа для выполнения математических операций.
  • Синус (sin): Функция, возвращающая синус угла.
  • Косинус (cos): Функция, возвращающая косинус угла.
  • Тангенс (tan): Функция, возвращающая тангенс угла.
  • Арксинус (asin): Функция, возвращающая обратный синус.
  • Арккосинус (acos): Функция, возвращающая обратный косинус.
  • Арктангенс (atan): Функция, возвращающая обратный тангенс.
  • Корень (sqrt): Вычисляет квадратный корень.
  • Натуральный логарифм (ln): Вычисляет натуральный логарифм.
  • Десятичный логарифм (log): Вычисляет десятичный логарифм.
  • Введение числа X: Предоставляет писать в выражении X и вводить его значение

График:

  • Присутствует кнопка графика, она активна, когда выражение валидно и присутствует X в выражении

Основные функции кредитного калькулятора:

  • Присутствует выбор вида кредита
  • Рассчет ежемесячного платежа: Функция, принимающая входные данные, такие как сумма кредита, процентная ставка и срок кредита, и возвращающая ежемесячный платеж по кредиту
  • Рассчет общей переплаты: Функция, вычисляющая общую сумму переплаты по кредиту, начисленные проценты.
  • Рассчет общей суммы кредита: Функция, вычисляющая общую сумму кредита, включая начисленные проценты и изначальную сумму кредита.
